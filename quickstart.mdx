## 1. Get your credentials

Generate an API key and secret from the [dashboard](https://xeniosproject.com).
See [Authentication](/authentication) for how requests are signed.

## 2. Call the gateway

<CodeGroup>

```bash cURL
curl https://api.xeniosproject.com/v1/account/wallets \
  -H "Authorization: Bearer $XENIOS_API_KEY"
```

```ts TypeScript
const res = await fetch("https://api.xeniosproject.com/v1/account/wallets", {
  headers: { Authorization: `Bearer ${process.env.XENIOS_API_KEY}` },
});
const wallets = await res.json();
```

</CodeGroup>
