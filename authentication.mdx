---
title: "Authentication"
description: "How to authenticate requests to the Xenios gateway."
---

All requests to the gateway must be authenticated. Send your API key as a bearer
token in the `Authorization` header.

```bash
Authorization: Bearer <YOUR_API_KEY>
```

<Warning>
  Never expose your API secret in client-side code or commit it to version
  control. Keep it in a secret manager or environment variable.
</Warning>

## Errors

Unauthenticated or malformed requests return a `401 Unauthorized`. See the
[API Reference](/api-reference) for the full error schema.
